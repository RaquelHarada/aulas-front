Box Model do CSS

Definição

O Box Model (Modelo Caixa) é um conceito fundamental no CSS que define como os elementos são dimensionados e posicionados na página.

Nos exemplos abaixo, amabas as caixas possuem width:300px no CSS. Mas, na prática, suas dimensões finais são diferentes.
