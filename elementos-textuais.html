Front-End

Área de trabalho em que produzimos o visual, ou seja, interface de aplicações web.

Back-End

Área de desenvolvimento de sistemas, ou seja, que envolve a manuipulação de dados dinâmicos.

Nesta área é comum o trabalho com linguagens de programação de servidor.

Exemplos de tecnologias : PHP, Python e Java.

Importante: existem várias outras tecnologias de back-end.
