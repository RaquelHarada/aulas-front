Explicando funções do CSS

min( ), max( ), calc( ), clamp( )

Funções utilizadas para criações de sites responsivos, criando um design fluído que se adeque a necessidade do usuário.

min( )

É utilizado para limitar o tamanho máximo de algo, sendo assim não permitindo que o valor proposto seja ultrapassado.

Exemplo:width: min(90%, 900px); 90% se refere ao elemento PAI, tamanho da tela sendo flexível, e o 900px é o limite máximo.


max( )

Diferente do min(), a função max() é utilizada para limitar o tamanho mínimo, selecionando o maior tamanho entre dois ou mais valores.

Exemplo:padding: max(1rem, 2vw); 1rem serve como o mínimo seguro, e 2vw vai crecer conforme a tela aumenta.


calc( )

Utilizado para realizar cálculos matemáticos básicos, (+, -, *, /), podendo misturar diferentes unidades de medida (%, px, vw, etc.)

Exemplo: padding: calc(1rem + 1vw); 1rem garante que padding nunca fique pequeno demais, 1vw fica proporcional ao tamanho da tela.


clamp( )

Função que seleciona um valor entre o min, max e o ideal, assim retornando ao valor ideal

Exemplo: font-size: clamp(1rem, 3vw, 2rem); Começa no valor ideal (3vw) mínimo 1rem e máximo 2rem. Se passar de 32px → trava em 32px
